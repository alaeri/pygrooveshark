class grooveshark.core.client.Client(*args, **kwargs)¶

A client for Grooveshark’s API which supports:

  • radio (songs by genre)
  • search for songs, artists and albums
  • popular songs
Parameters:
  • session – stored session information as returned by init_session() method.
  • token – stored token information as returned by init_token() method.
  • queue_id – stored queue id as returned by init_queue() method.
  • proxies – dictionary mapping protocol to proxy.
init()¶

Fetch Grooveshark’s session and token.

Return type:tuple: (init_session(), init_token(), init_queue())
init_queue()¶

Initiate queue. Make sure to call init_session() and init_token() first.

Return type:queue_id

You can store the returned queue_id and use it again over the queue_id argument of the Client class.

init_session()¶

Fetch Grooveshark’s session.

Return type:tuple: (session, secret, country, user)

You can store the returned tuple and use it again over the session argument of the Client class.

init_token()¶

Fetch Grooveshark’s communication token. Make sure to call init_session() first.

Return type:tuple: (token, token_time)

You can store the returned tuple and use it again over the token argument of the Client class.
